📋 WHAT IS IT?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
An intelligent system that detects irrelevant queries and redirects
students to focus on placement preparation with personalized messages.

🎯 KEY FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ Relevance scoring (0-100%)
✓ Off-topic detection (30% threshold)
✓ Personalized redirects based on performance
✓ Study suggestions
✓ Cost savings (blocks before API calls)

⚙️ CONFIGURATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Adjust Threshold (in intelligent_cutoff.py):
  • 40% = Stricter (blocks more queries)
  • 30% = Balanced (default, recommended)
  • 20% = Lenient (blocks only obvious off-topic)

Add Keywords (in intelligent_cutoff.py):
  • high_relevance: assessment, placement, coding...
  • low_relevance: game, movie, party...
